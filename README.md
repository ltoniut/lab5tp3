- Se usó una base de datos mysql debido a su compatibilidad mainstream con JDBC, el sistema para conexiones de bases de datos
con Java más popular y documentado que pude encontrar. (Con el beneficio adicional de ser el Framework a utilizar en
Programación Avanzada en el cuatrimestre corriente)
- El juego es sencillo: Se toman 3 jugadores (En el trabajo, Leandro, Franco y Antonino) y se los asocia con un repartidor (dealer) que
repartirá cartas hasta que se le acabe el mazo. Todos los jugadores compiten por recibir las cartas entrantes. Al terminar el juego, se
suman los números en las cartas de cada mano y el vencedor es quien tenga la mayor suma.
